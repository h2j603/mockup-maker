# Mockup Maker

영상 + 디바이스 프레임 PNG 합성 → MP4 내보내기 웹 앱

## 사용법
1. `frames/` 폴더에서 디바이스 프레임 PNG 선택
2. 영상 업로드
3. 플랫폼 선택 후 내보내기

## frames 폴더
디바이스 프레임 PNG를 여기에 넣어두세요.  
스크린 부분이 **투명(알파)** 처리된 PNG 파일이어야 합니다.

무료 프레임 소스:
- [Figma Community](https://www.figma.com/community)
- [Mockuphone](https://mockuphone.com)
